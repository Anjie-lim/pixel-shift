# Pixel Shift

Interactive Shot Detection Sandbox

## Features
- Pixel-based cut detection engine
- Human vs AI comparison system
- Real-time Chart.js visualization
- Adjustable sensitivity tuning
- Evaluation system (TP / FP / FN / F1)
- GitHub Pages ready (no backend)

## How to Run
1. Upload to GitHub repo
2. Enable GitHub Pages
3. Open index.html via Pages URL

## Controls
- Spacebar → mark human cut
- Slider → adjust sensitivity
- Buttons → switch clips / evaluate

## Tech Stack
- Vanilla JavaScript (ES6+)
- HTML5 Video + Canvas API
- Chart.js (CDN)
